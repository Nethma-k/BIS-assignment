# BIS-assignment
This project presents a complete data analytics workflow applied to a real-world marketing dataset. The objective is to identify key drivers of customer campaign response and determine how customer segments  respond to marketing campaigns and other customer demographic features that contribute to consumer spending. 

Business Question answered in the analysis - How do life-stage factors (Age, Income, and Family Composition) influence customer behaviour, and how do these segments respond to marketing campaigns?

The analysis follows a structured data science process, including:
Data preprocessing
Exploratory Data Analysis (EDA)
Statistical analysis

Dataset: Marketing Campaign Dataset
Total Records: 2,240 customers

Key Features Included:
Demographics: Age, Income, Education, Marital Status
Family Composition: Number of children/teenagers
Spending: Wines, Fruits, Meat, Fish, Sweets, Gold
Engagement: Recency, Website visits
Campaign Response: Acceptance indicators for 5 campaigns
